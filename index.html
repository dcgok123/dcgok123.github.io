帝国CMS一键安装
想到即可做到 - 帝国CMS
服务器套件所包含的软件：
APACHE 2.2.4
PHP 5.2.0
MYSQL 5.0.27
ZEND Optimizer 3.2.6
phpMyAdmin-2.9.1.1
帝国CMS7.5开源版(GBK)
安装步骤
1、解压安装包；
2、将安装包的“EmpireServer”目录复制到D盘根目录；(D:\EmpireServer目录名不可更改)
3、双击“D:\EmpireServer\一键安装.bat”；(用windows7或windows10及以上则要鼠标右键以管理员身份运行)
4、至此，运行环境及帝国CMS全部安装完毕；
5、在浏览器打入：http://localhost/e/admin 后回车，进入帝国CMS后台登陆界面。
后台登陆用户名、密码与认证码均为admin
(无需再安装帝国CMS，可直接登陆后台进行管理)
 
注意：如果服务器已经安装了IIS（asp运行环境），安装此软件包就需要将IIS的端口号改为其它，或者关闭IIS。
或者使用另一种方法：修改apache的80端口，用记事本打开“D:\EmpireServer\编辑APACHE配置文件”找到：“Listen 80”将80改为其它端口，如“Listen 8080”。改变端口后访问要用：http://localhost:端口/访问，如：http://localhost:8080/。
卸载套件
1、双击“D:\EmpireServer\一键卸载.bat”；
2、删除“D:\EmpireServer”目录即可完全卸载。
地址说明
帝国CMS管理后台：http://localhost/e/admin
后台登陆用户名、密码与认证码均为admin
 
phpMyadmin（管理mysql 数据库）地址：http://localhost/phpmyadmin
mysql 数据库用户名为root，密码为空
目录结构
D:\EmpireServer\php ：APACHE+PHP+MYSQL+ZEND安装目录
D:\EmpireServer\web ：网站目录 (PHP运行目录，安装其他PHP程序复制到这个目录即可。)
